# 🛠️ ZH-Skills

**一个理工科背景的人，用九年时间在四个行业之间反复跨域之后，发现自己最真实的需求是：**  
不要泛化的 AI 工具，要能精准锚定自己思维方式的个人认知工具。

这个仓库是我为自己编写的 Claude Skill 库。每一个 Skill 都来自一个具体的痛点——不是"AI 能力不够"，而是"现有的 AI 工具不懂我在做什么"。

---

## 我的核心需求与痛点

我的阅读和思考习惯是**纵向深钻而非横向扫描**——我不需要书摘，我需要找到一本书在认知史上真正挪动了什么；我不需要摘要，我需要在两本书的交叉沉默处找到下一个真正值得问的问题。

同时，我对**信息茧房和单一叙事高度警惕**——特别是东亚语境里对人物和观念的美化压缩。我需要能切换视角、以真实异质眼光重新打量熟悉事物的工具。

这两个痛点是这个库所有 Skill 的底层动机。
